# Integrating Templates Engines

Seagull is structured so it's easy to integrate the template engine of your choice.  Currently implemented are:

 * [Flexy][1] (default)
 * [Smarty][2]
 * [Savant2][3]

Also see the [Howto/Templates][4] section.
